# AI-Based Sentiment Analysis on Amazon Product Reviews

This repository contains the project files for CDS6344 Social Media Computing. The project focuses on analyzing Amazon product reviews using Natural Language Processing techniques. The reviews were classified into Negative, Neutral and Positive sentiments.

## Project Overview

The project includes data preprocessing, traditional machine learning models, deep learning models, transformer-based models, Aspect-Based Sentiment Analysis and data visualization. The main purpose of this project is to understand customer opinions from Amazon reviews and compare the performance of different sentiment analysis models.

## Repository Structure

* `data/`
  Contains the dataset files used in this project.

* `notebooks/`
  Contains all project code in Jupyter Notebook format.

* `outputs/`
  Contains visualization outputs, model result files, confusion matrices, word clouds and ABSA charts.

## Dataset

The dataset used in this project is the Amazon Reviews Dataset from Kaggle.

Dataset source: [Amazon Reviews Dataset on Kaggle](https://www.kaggle.com/datasets/dongrelaxman/amazon-reviews-dataset?resource=download)

The dataset was cleaned and preprocessed before being used for sentiment analysis. The cleaned dataset contains review text and sentiment labels, while the preprocessed dataset contains the processed text used for model training and evaluation.

## Models Used

The models used in this project include:

* Logistic Regression
* Naive Bayes
* Support Vector Machine
* LSTM
* BiLSTM
* BERT
* DistilBERT
* RoBERTa

## Results

The results show that transformer-based models performed better than traditional machine learning and deep learning models. RoBERTa achieved the best performance among the models tested. The visualizations also show that Negative reviews were the largest sentiment class in the dataset.

## Aspect-Based Sentiment Analysis

Aspect-Based Sentiment Analysis was used to identify customer opinions based on specific aspects such as Customer Service, Delivery, Packaging, Price / Value and Product Quality. The ABSA results showed that Customer Service, Delivery and Price / Value received many negative mentions.

## Note on Model Files

Large trained model files such as `.pt` files are not included in this repository because of file size limits. The model results and visual outputs are included in the `outputs/` folder and the models can be regenerated by running the notebooks.

## Future Work

For future work, the project can be improved by using a more balanced dataset so the models can learn better from negative, positive, and neutral reviews. More labelled neutral reviews can also be added because neutral sentiment was the hardest class to classify in this project. The system can also be improved by deploying an online dashboard, collecting reviews in real time, supporting multilingual reviews and testing the model on other e-commerce review datasets.
