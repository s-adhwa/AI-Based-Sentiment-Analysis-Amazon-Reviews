# AI-Based Sentiment Analysis on Amazon Reviews

## Project Overview

This project focuses on sentiment analysis of Amazon product reviews using Natural Language Processing techniques. The system applies multiple approaches including traditional machine learning, deep learning, transformer-based models, and aspect-based sentiment analysis to classify reviews into negative, neutral, and positive sentiments.

The main goal is to understand customer opinions and compare the performance of different models on real-world review data.

## Dataset

The dataset used in this project is the Amazon Reviews Dataset from Kaggle:

https://www.kaggle.com/datasets/dongrelaxman/amazon-reviews-dataset

It contains product reviews along with rating information used for sentiment classification.

## Repository Structure

* `data/`
  Contains the dataset files used in this project.

* `notebooks/`
  Contains all project code in Jupyter Notebook format.

* `outputs/`
  Contains visualization outputs, model result files, confusion matrices, word clouds and ABSA charts.

## Dataset

The dataset used in this project is the Amazon Reviews Dataset from Kaggle.

Dataset source: [Amazon Reviews Dataset on Kaggle](https://www.kaggle.com/datasets/dongrelaxman/amazon-reviews-dataset?resource=download)

The dataset was cleaned and preprocessed before being used for sentiment analysis. The cleaned dataset contains review text and sentiment labels, while the preprocessed dataset contains the processed text used for model training and evaluation.

## Models Used

The models used in this project include:

* Logistic Regression
* Naive Bayes
* Support Vector Machine
* LSTM
* BiLSTM
* BERT
* DistilBERT
* RoBERTa

## Results

Model evaluation results show a clear advantage of transformer-based approaches over both traditional machine learning and deep learning methods. RoBERTa consistently produced the strongest performance among all tested models. In addition, the dataset distribution highlights that negative reviews dominate the data, indicating that dissatisfaction is more prevalent compared to other sentiment categories.

## Note on Model Files

Large trained model files such as `.pt` files are not included in this repository because of file size limits. The model results and visual outputs are included in the `outputs/` folder and the models can be regenerated by running the notebooks.

## Future Work

For future work, the project can be improved by using a more balanced dataset so the models can learn better from negative, positive, and neutral reviews. More labelled neutral reviews can also be added because neutral sentiment was the hardest class to classify in this project. The system can also be improved by deploying an online dashboard, collecting reviews in real time, supporting multilingual reviews and testing the model on other e-commerce review datasets.
